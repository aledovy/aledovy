# Aledovy

## About Me

Hi, I'm **Dovydas**, a bonafide security analyst specializing in detecting Indicators of Compromise (IOCs) in phishing, scam, and malware websites. My passion is automating threat detection and helping organizations stay ahead of cyber threats.

## Project Overview

**Aledovy** is a collection of tools and scripts designed to automate the detection and analysis of malicious websites. The goal is to streamline the process of identifying IOCs and provide actionable intelligence for security teams.

## Features
- Automated IOC detection
- Phishing, scam, and malware site analysis
- Easy integration with security workflows
- Customizable and extensible scripts

## Getting Started
1. Clone this repository
2. Review the available scripts and tools
3. Follow the instructions in each script to run analyses
